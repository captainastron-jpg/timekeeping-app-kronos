
  # Time Keeping Web App

  This is a code bundle for Time Keeping Web App. The original project is available at https://www.figma.com/design/nNcRSxCOHJCprndj4Vsr9z/Time-Keeping-Web-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Server environment

  Create a `.env` file in the `server/` folder with values like:

  ```env
  MYSQL_HOST=localhost
  MYSQL_USER=root
  MYSQL_PASSWORD=your_password
  MYSQL_DATABASE=time_keeping_app
  MYSQL_PORT=3306
  JWT_SECRET=your_jwt_secret
  ```
